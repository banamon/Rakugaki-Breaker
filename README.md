# 落書き崩し
## システム概要
大学2年生時に受講した演習授業において開発した対戦型のブロック崩しです．

「対戦型のゲームの開発」というテーマで，授業資料に沿ってオセロを開発したのちに，
ターン性ではなく，リアルタイムのゲームを開発したいと思い，
対戦型のブロック崩しのルールなどを考案し，開発を行いました
